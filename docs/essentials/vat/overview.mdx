The Vat API can be used to calculate VATs on goods, validate a company's VAT numbers, and get the latest VAT rates for specific goods in a country.
In `abstract-python`, the `VatAPI` class is used to communicate with this API.

## Supported currency codes
AbstractAPI supports ISO 4217 currency codes for the VAT API which can be found [here](https://docs.abstractapi.com/vat-validation#country-codes)


## Get currency codes

With `abstract-python` you can get the supported country codes as a Python List. See usage below:

```python
from abstract_python.abstract import VAT_SUPPORTED_COUNTRIES

print(VAT_SUPPORTED_COUNTRIES)
```

## Use case

If your Python app allows users to enter country codes themselves, you may need to validate user inputs before querying the `VatAPI` class. 

See usage below:

```python
from abstract_python.abstract import VAT_SUPPORTED_COUNTRIES

base_currency = input("Enter a country code")

if base_currency in VAT_SUPPORTED_COUNTRIES:
    # Query an endpoint in the VatAPI class 
```
